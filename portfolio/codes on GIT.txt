ls- a = shows all files in the current directory
git init= initializes the current directory as a an empty git repository
git add= adds all files inside the directory from "working directory" to "staging".
git commit -m "<my first ever commit>"
	// commit should be descriptive
	// should be in the present tense

git config --global user.email "you@example.com"
git config --global user.name "Your Name"


git status -s //will display changes
git log //will display commits/versions
git remote add <remote name> <github URL> 
git remote -v
git push remote branch = makes a copy to the github server
